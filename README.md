# 📊 Marketing Campaign Performance & A/B Testing Analysis

---

## 📌 Overview
This project analyzes marketing campaign performance to identify effective strategies using data analysis and A/B testing.  
The goal is to support data-driven decision-making in marketing.

---

## 🎯 Business Problem
Companies invest heavily in marketing campaigns but often lack clarity on which strategies deliver the best results.

---

## ⚙️ Approach
- Data Cleaning & Preparation (Python)  
- Feature Engineering (CTR, Cost Efficiency)  
- Exploratory Data Analysis (EDA)  
- A/B Testing (Statistical Analysis)  
- Business Recommendations  

---

## 📂 Dataset
### 📥 How to Use

1. Download or clone this repository  
2. Extract the ZIP file  
3. Place the extracted CSV file in the project directory  
4. Update the file path in the notebook if necessary 

---

## 📊 Key Insights
- Campaign performance is consistent across types and channels  
- No strong relationship between cost and ROI  
- Engagement (CTR) varies significantly across campaigns  

---

## 🧪 A/B Testing Results
- No statistically significant difference between campaigns  
- Performance differences are minimal  
- Optimization should focus on execution rather than channel selection  

---

## 💡 Business Recommendations
- Improve campaign creatives and targeting  
- Focus on engagement optimization  
- Avoid increasing budget without performance validation  
- Continue experimentation through A/B testing  

---

## 📈 Conclusion
This project demonstrates that marketing success depends more on optimization and execution than on simply choosing different channels or campaign types.
